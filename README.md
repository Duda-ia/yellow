# Oce Yellow Engenharia - Site Institucional

Um site institucional minimalista e sofisticado para a Oce Yellow Engenharia, inspirado na elegância da Construtora Gamboa.

## 🎨 Design e Inspiração

O design foi inspirado na **Construtora Gamboa**, incorporando:

- **Layout minimalista** com foco na simplicidade
- **Tipografia fina e espaçada** para elegância
- **Paleta sóbria e elegante** com tons neutros
- **Ampla área de respiro** entre seções
- **Animações sutis** que não distraem do conteúdo

## 🎯 Características

### Design
- **Paleta de cores**: 
  - Fundo: #111111 (preto suave)
  - Texto: #EAEAEA (branco suave)
  - Detalhes: #FFD95B (amarelo claro pastel) e #CFCFCF (cinza quente)
- **Tipografia**: 
  - Fonte principal: Space Grotesk (sans-serif moderna)
  - Hierarquia tipográfica com pesos 300-700
  - Letter-spacing generoso para elegância
- **Layout**: Mobile-first, responsivo e com espaçamento generoso
- **Animações**: Transições leves de fade-in e hover

### Funcionalidades
- ✅ Navegação fixa e discreta
- ✅ Hero section com headline elegante

- ✅ Galeria de portfólio com hover colorido
- ✅ Seção de parceiros com logos
- ✅ Design totalmente responsivo

- ✅ Otimizado para SEO

## 🚀 Como Usar

### Instalação Local
1. Clone ou baixe os arquivos do projeto
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. O site estará funcionando completamente offline

### Desenvolvimento
Para fazer alterações:

1. **HTML**: Edite `index.html` para modificar estrutura e conteúdo
2. **CSS**: 
   - `css/style.css` - Estilos principais
   - `css/responsive.css` - Estilos responsivos
3. **JavaScript**: `js/main.js` - Funcionalidades interativas

### Personalização

#### Cores
As cores principais estão definidas no CSS:
```css
--background-color: #111111;    /* Preto suave */
--text-color: #EAEAEA;          /* Branco suave */
--accent-color: #FFD95B;        /* Amarelo claro pastel */
--secondary-color: #CFCFCF;     /* Cinza quente */
```

#### Conteúdo
- **Logo**: Substitua "OCE YELLOW" e "ENGENHARIA" no HTML
- **Imagens**: Substitua os placeholders por imagens reais
- **Textos**: Edite os textos diretamente no HTML
- **Contatos**: Atualize as informações de contato no footer
